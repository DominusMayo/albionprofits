Для разработчиков имеется API для получения в json формате вместо графического интерфейса.
В данный момент имеется только одна точка для получения выгодных сделок между двумя городами.

Выгодные сделки между двумя городами(JSON):
https://albionprofits.tk/api/albion/prices/cities?format=json&locations=Thetford,Martlock&category_items=mellee_sword&tier=T4&chart=0&chart=%401&chart=%402&profit=&hours=

Параметры:

  • locations: по очереди город закупа, город продажи
  • category_items: категория предметов для поиска. Подробнее ниже.
  • tiers: тир предметов.
  • chart: зачарование для предметов.
  • profit: Ожидаемая прибыль для одной продажи.
  • hours: Актуальность информации.

    • Категория предметов:

  • mellee_sword: Мечи
  • range: Оружие дальнего боя
  • mellee_mace: Булавы
  • mellee_staff: Шесты
  • mellee_hammer: Молоты
  • mellee_daggers: Кинжалы
  • mellee_axe: Топоры
  • staff: Посохи/Магия
  • plate_shoes: Латные ботинки
  • plate_head: Латные шлемы
  • plate_armor: Латные доспехи
  • leather_shoes: Кожанные ботинки
  • leather_head: Кожанные капюшоны
  • leather_armor: Кожанные доспехи
  • cloth_shoes: Тканевые ботинки
  • cloth_armor: Тканевые доспехи
  • cloth_head: Тканевые колпаки
  • capes: Плащи
  • bags: Сумки
  • luxury: Роскошь
  • resources: Ресурсы
